# Upload Guide

## Recommended repository name

```text
hypas-redm-anticheat
```

Expected URL:

```text
https://github.com/hypasscripts/hypas-redm-anticheat
```

## Create the repository with

- Visibility: **Public**
- Add README: either option is acceptable
- `.gitignore`: **None**
- Licence: **No licence**

## Upload this package

1. Extract the ZIP on your computer.
2. Open the new repository.
3. Select **Add file**.
4. Select **Upload files**.
5. Drag all extracted files and folders into the upload area.
6. Commit with:

```text
Add public Hypas Anti-Cheat documentation
```

## Add screenshots

Upload your screenshots into the `images` folder using the filenames referenced in `README.md`, or edit the README to match your filenames.

## Replace placeholders

Search the repository for:

```text
ADD_YOUR_TEBEX_PRODUCT_URL_HERE
ADD_YOUR_DISCORD_INVITE_HERE
ADD_A_SECURITY_EMAIL_HERE
```

Replace each placeholder before publicly promoting the repository.

## Enable GitHub Pages

Read:

```text
docs/github-pages-setup.md
```

## Do not upload

- `client.lua`
- `server.lua`
- Detection code
- SQL containing sensitive implementation details
- Escrowed files
- Customer files
- Licence data
- Webhook URLs
- Internal exploit research
