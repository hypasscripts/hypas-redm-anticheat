# Support

## Before requesting support

Confirm that:

1. The resource folder name is correct.
2. The supplied SQL has been imported.
3. `oxmysql` and the required framework resources start before Hypas Anti-Cheat.
4. The resource is included in `server.cfg`.
5. Your administrator account has the required ACE permission.
6. You have restarted the server after installation or structural configuration changes.
7. You have reviewed `/achealth` and `/acaudit`.

## Information to provide

- Hypas Anti-Cheat version
- RedM artifact/server build
- VORP Core version
- oxmysql version
- Relevant console error
- Exact command or action that failed
- Whether the problem occurs for all staff or one account
- Whether the issue started after a configuration change

## Privacy

Remove:

- Database passwords
- Tebex licence details
- Player identifiers that are not necessary
- Discord webhook URLs
- Private exploit payloads
