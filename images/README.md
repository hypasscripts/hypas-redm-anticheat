# Screenshot folder

Upload your product screenshots here.

Recommended filenames:

```text
dashboard.png
player-investigation.png
evidence.png
logs.png
health.png
settings.png
```

Use PNG or WebP where possible.

Recommended screenshot rules:

- Remove player personal information.
- Remove licence keys and webhook URLs.
- Use a consistent resolution.
- Crop unnecessary desktop or browser content.
- Keep the panel readable.
- Use descriptive alternative text in Markdown.
