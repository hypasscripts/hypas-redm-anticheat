# Contributing

This is a documentation repository for a commercial RedM security product. It does not accept protected source-code contributions.

Contributions are welcome for:

- Documentation corrections
- Typographical fixes
- Clearer installation instructions
- Framework compatibility notes
- Reproducible documentation issues
- Suggestions for public educational guides

Do not submit:

- Commercial source code
- Decrypted or decompiled files
- Detection bypasses
- Exploit payloads
- Customer licence information
- Private server credentials or logs containing personal data

Open a documentation issue or submit a pull request containing only the relevant documentation change.
